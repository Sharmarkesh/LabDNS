# DNS  Lab

Buy a domain, deploy NGINX on an EC2 instance, and make the page load over your own domain.
Documentation of my DevOps learning journey, covering networking fundamentals and hands-on practice.

## Environment

- **Local:** Accessing EC2 through SSH my windows machine
- **Remote:** AWS, Cloudflare


## Labs

| Lab | Description |
|---|---|
| Nginx on EC2 with Cloudflare DNS| Deploy an Nginx web server on AWS EC2 and point a custom domain Djibsec to it via Cloudflare DNS |

## Skills Demonstrated

- OSI & TCP/IP models and layer-based troubleshooting
- IP addressing, subnetting, and CIDR notation
- Core protocols: TCP, UDP, HTTP/S, SSH, DNS, DHCP
- Routing fundamentals and static/dynamic routing
- Firewalls, security groups, and network ACLs
- NAT, port forwarding, and load balancing
- Container networking (Docker/Kubernetes)
- Cloud networking concepts
- Network troubleshooting with `ping`, `traceroute`, `dig` and nslookup
